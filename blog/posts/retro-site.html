Building a Retro Personal Site in 2025

Posted on February 5, 2025

This post walks through rebuilding a nostalgic 2000s-style blog with modern CSS variables, flex layouts, and a sprinkle of JavaScript for counters and auto-generated archives. The goal was to keep the charm of old blogs while using today’s standards.

Highlights:

  • CSS variables for theming and quick tweaks
  • Flexbox for the two-column layout that keeps the sidebar in place
  • A small JS router to swap main content without touching the sidebar
  • Automatic monthly archives driven by data-date attributes
